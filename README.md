Repositorio enfocado en el aprendizaje práctico de Python, donde desarrollo ejercicios, retos y pequeños proyectos aplicando lógica de programación, estructuras de datos y resolución de problemas.

Este contenido está basado en el curso **Introducción a la Inteligencia Artificial (80h)** de [Fundación Somos F5](https://www.somosf5.org/) y en práctica adicional mediante retos auto-propuestos.

## 🛠️ Tecnologías utilizadas

- Python
- Git & GitHub
- Visual Studio Code

### Estudiando_Python_1 🐍

Apuntes estructurados y ejemplos prácticos sobre:

- Variables y tipos de datos  
- Condicionales  
- Bucles  
- Funciones
- Vectores y Matrices 

### Menu_burger_app 🍔​

Pequeña aplicación interactiva para la compra de hamburguesas con selección de extras.

🔗 Demo: https://menu-burger-app.lovable.app

Conceptos aplicados:
- Entrada de datos por usuario  
- Estructuras condicionales  
- Lógica de decisión  

### Retos 🎯​

Actividades prácticas propuestas para reforzar el aprendizaje.

Incluyen:
- Ejercicios de lógica  
- Resolución de problemas  
- Práctica de sintaxis en Python

## 🎯 Objetivo

Consolidar conocimientos en Python mediante práctica continua y construir una base sólida orientada al desarrollo en Data Analysis e Inteligencia Artificial.
